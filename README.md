A free, hands-on course on the Microsoft Agent Framework in C#.

We start from the fundamentals and incrementally build out a fully featured AI agent called Iron Mind AI -a personal trainer agent that gains new capabilities with every instalment.

The course covers conversations and threads, function tools, agents as function tools, human-in-the-loop approval, contextual memory, background responses, MCP servers, email marketing integration, and RAG using vector stores.

Each video is accompanied by a detailed blog post and full source code at https://jamiemaguire.net

No prior experience with Semantic Kernel or AutoGen is required. Everything you need is covered.
